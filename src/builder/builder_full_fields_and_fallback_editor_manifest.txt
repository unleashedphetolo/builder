Builder full fields and fallback editor update

Updated:
- Builder.jsx
- section-editor/BuilderSectionEditor.jsx
- section-editor/editorRegistry.js

What changed:
- When an edit button opens a saved section that only has a title, Builder.jsx now merges the full fallback content from the clicked template section into the saved section before opening the editor.
- Existing saved values are preserved.
- Missing arrays, bullets, numbered items, PDF links, image fields and nested content are restored from the template fallback.
- BuilderSectionEditor now has a dynamic template-field editor for rich school pages that do not have a dedicated hard-coded editor yet.
- Dynamic editor supports text fields, textarea fields, text-list bullets, object collections, nested list fields and JSON-style object fields.
- Media tab still discovers image and document fields automatically.
